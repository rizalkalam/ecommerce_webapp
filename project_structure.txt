# Penjelasan Inisiasi Komponen & Page
- Buat komponen di folder src/components jika elemen UI tersebut akan digunakan ulang di beberapa tempat (misal: Navbar, Footer, Card, dsb).
- Buat page di folder src/pages untuk setiap halaman utama aplikasi (misal: HomePage, ProductPage, CartPage, dsb). Page biasanya mewakili satu route.
- Jika ada fitur besar atau section yang kompleks, buat folder baru di dalam components atau pages untuk mengelompokkan file terkait.
- Selalu mulai dari membuat komponen kecil yang reusable, lalu susun menjadi komponen lebih besar atau page.
- Dokumentasikan setiap komponen/page secara singkat di awal file agar memudahkan kolaborasi.
